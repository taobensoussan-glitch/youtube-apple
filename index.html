Configuration Google

Pour afficher les vraies vidéos, ajoutez vos identifiants Google (gratuit, ~5 min).

  1. Allez sur Google Cloud Console
  2. Créez un projet, activez YouTube Data API v3
  3. Créez une Clé API (Identifiants → Créer)
  4. Créez un ID client OAuth type "Application Web"
  5. Origine JS autorisée : votre URL
